# GeoViz — 全球电厂交互地图

基于 [全球电厂数据库](https://github.com/wri/global-power-plant-database) 的交互式地图仪表盘，可视化全球约 35,000 座电厂，支持按能源类型、国家、装机容量筛选。

**在线访问：** 部署于 Vercel

---

## 技术栈

React 18 + TypeScript · deck.gl · MapLibre GL · TailwindCSS · Vite

## 本地运行

```bash
npm install
npm run dev
```
